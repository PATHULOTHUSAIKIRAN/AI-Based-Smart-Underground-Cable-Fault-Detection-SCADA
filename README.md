# AI-Based Smart Underground Cable Fault Detection & SCADA Monitoring System

## Overview

This project presents an AI-enabled Industrial IoT system for detecting underground cable faults using ESP32 and a Python-based SCADA platform. The system continuously monitors three-phase electrical parameters, detects various fault types, estimates fault location, and provides real-time monitoring through a web dashboard.

---

## Features

- Real-time Underground Cable Fault Detection
- LG, LL, LLG, LLL and LLLG Fault Classification
- Fault Distance Estimation
- ESP32 Industrial IoT Controller
- Live SCADA Dashboard
- AI-based Fault Analysis
- Predictive Health Monitoring
- Automatic & Manual Relay Control
- Voice Alert System
- CSV Report Generation
- Event Logging
- Browser-Based Monitoring
